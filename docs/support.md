# Підтримати проєкт

**OpenFPV** — волонтерський проєкт української спільноти OpenIPC.

Ми безкоштовно ведемо документацію, перекладаємо гайди та тестуємо прошивки. Ваша підтримка допомагає проєкту жити й розвиватися.
