# Прошивки камер

## Офіційні прошивки OpenIPC

Основна прошивка для всіх підтримуваних камер. Завантажити з [openipc.org](https://openipc.org/).

## APFPV

Популярна кастомна прошивка від спільноти з покращеними FPV параметрами.

- Оптимізовані налаштування для мінімальної затримки
- Розширені опції відео

## APALink

Прошивка для систем з Adaptive-Link.

- Автоматичне підлаштування потужності сигналу
- Збереження зв'язку на межі дальності

## Greg's Firmware

Альтернативна прошивка від розробника спільноти.

## Оновлення прошивки

Рекомендований спосіб — через [Multiconfigurator](/configuration/multiconfigurator).

::: warning
Перед оновленням збережіть поточні налаштування камери.
:::
