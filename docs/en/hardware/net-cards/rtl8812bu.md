# RTL8812BU

An alternative 5V Wi-Fi card to the RTL8812EU2.

| Parameter | Value |
|-----------|-------|
| Voltage | 5V |
| Max Frequency | 40 MHz |
| Interface | USB |

> Detailed documentation is in development.
