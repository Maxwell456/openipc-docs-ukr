A digital HD receiver based on OpenIPC / Ruby FPV, designed for stable, high-quality video transmission from the drone to the ground station.

---

### Technical specifications

| Parameter                     | Value                                                            |
|-------------------------------|------------------------------------------------------------------|
| 📡 Frequency range            | 5180–5885 MHz                                                   |
| 📶 Transmit power             | <25 dBm (FCC), <14 dBm (CE), <20 dBm (SRC), <25 dBm (MIC)       |
| 🎥 Video output               | Mini HDMI (1080P 60fps / 720P 60fps)                            |
| 💾 Internal memory            | 32 GB (system files included)                                   |
| 🔌 Interfaces                 | Mini-HDMI, MicroSD, DC-in (9–30V), Type-C (OTG)                 |
| 📐 Enclosure dimensions       | 110 × 27.3 × 46 mm                                              |
| ⚖️ Weight                     | 122 g (without antenna)                                         |
| ⚙️ Compatibility              | OpenIPC, Ruby FPV                                               |
