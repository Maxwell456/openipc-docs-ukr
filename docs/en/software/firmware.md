# Camera Firmware

## Official OpenIPC Firmware

The base firmware for all supported cameras. Download from [openipc.org](https://openipc.org/).

## APFPV

A popular community custom firmware with improved FPV parameters.

- Optimized settings for minimal latency
- Extended video options

## APALink

Firmware for systems with Adaptive-Link.

- Automatic signal power adjustment
- Maintains connection at the edge of range

## Greg's Firmware

An alternative firmware from a community developer.

## Updating Firmware

The recommended method is via [Multiconfigurator](/en/configuration/multiconfigurator).

::: warning
Back up your current camera settings before updating.
:::
