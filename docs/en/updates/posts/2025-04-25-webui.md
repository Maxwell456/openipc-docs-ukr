## Web interface for camera settings

With the latest **Air Unit** firmware updates, **WebUI** support has been added, allowing direct access to camera settings through a browser.

::: warning Note!
**You must first update your camera firmware to use this feature.**
:::

**Accessing the interface**

Open one of the following addresses in your browser:

- [http://openipc.local](http://openipc.local)
- [http://192.168.1.10](http://192.168.1.10)

**Login credentials**

- **Username:** `root`
- **Password:** `12345`

---

**First login**

On the first login, the system will ask you to re-enter the password. After that it will be saved to grant full access to the camera web interface.

::: info Tip!
It is recommended to change the password after the first login for security (default: 12345).
:::
