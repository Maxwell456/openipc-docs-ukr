## 🔹 OpenIPC 4G/LTE

- OpenIPC supports 4G/LTE modems via USB or PCIe.
- For a stable connection, using WireGuard VPN over the mobile network is recommended.
- LTE allows controlling the drone over long distances and transmitting telemetry/video even beyond line of sight.
- Fallback mode is supported (e.g. switching to ELRS on LTE connection loss).
