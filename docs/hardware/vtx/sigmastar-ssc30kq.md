# Камера SSC30KQ + IMX335

<img src="/images/ssc30kq.jpg" alt="OpenIPC FPV камера на SigmaStar SSC30KQ" width="420" height="356"/>

**SSC30KQ** — це бюджетний одноядерний процесор (SoC) від SigmaStar, по суті спрощена версія [SSC338Q](/hardware/vtx/sigmastar-ssc338q) (1 ядро замість 2). У парі з 5-мегапіксельним сенсором **Sony IMX335** він став одним із найпопулярніших варіантів для **легких та дешевих OpenIPC FPV-камер**: готовий модуль (наприклад, Anjoy MCL12 / L12B) коштує приблизно **11 $**.

Завдяки низькому енергоспоживанню та невеликій вазі SSC30KQ добре підходить для **компактних дронів, "whoop"-ів та збірок 4G/LTE** ([QuadroFleet](/software/openipc-4g)). За якість 4K та максимальну продуктивність відповідає старший [SSC338Q](/hardware/vtx/sigmastar-ssc338q), а SSC30KQ — це про **мінімальну ціну та вагу**.

::: tip Коротко
- **Дешево** (~11 $ за готовий модуль) і **легко** — для маленьких збірок
- Сенсор **IMX335** дає до **1080p\@60** і навіть **720p\@120** для плавного відео
- Затримка близько **80 мс** (1080p\@60) та **60 мс** (720p\@60)
:::

## Характеристики

| Параметр | Значення |
|---|---|
| **Процесор (SoC)** | SigmaStar SSC30KQ, ARM Cortex-A7 (1 ядро) |
| **Сенсор** | Sony IMX335, 1/2.8″, 5 Мп, Starlight (нічна зйомка) |
| **Кодування** | H.265 / H.264, вбудоване аудіо в потік |
| **Бітрейт (FPV)** | від ~15 Мбіт/с H.265 (залежно від налаштувань) |
| **Затримка** | ~80 мс (1080p\@60), ~60 мс (720p\@60) |
| **Мережа** | Ethernet 100 Mbps |
| **Wi-Fi** | через зовнішній USB-адаптер ([мережеві карти](/hardware/network-cards)) |
| **Flash-пам'ять** | NOR 8/16 МБ або NAND (залежно від версії) |
| **Живлення** | DC 5V або 12V (залежно від плати) |
| **Підтримка OpenIPC** | Так (FPV, Majestic, WFB-NG) |

## Відеорежими IMX335

Драйвер IMX335 в OpenIPC підтримує кілька режимів — можна обрати між роздільною здатністю та плавністю:

| Роздільна здатність | Частота кадрів | Застосування |
|---|---|---|
| 1920×1080 | 60 к/с | максимальна деталізація |
| 1600×904 | 80 к/с | баланс чіткості та плавності |
| 1424×800 | 100 к/с | швидкий політ |
| 1280×720 | 120 к/с | максимальна плавність, перегони |

::: info Чим вище FPS — тим нижча затримка
Для динамічного польоту 720p\@120 відчувається набагато "чіткіше", ніж 1080p\@60, хоча роздільна здатність менша.
:::

## Розпіновка роз'ємів

<img src="/images/ssc30kq-pinout.jpg" alt="Розпіновка роз'ємів OpenIPC-камери SSC30KQ" width="420" height="420"/>

Типова розпіновка роз'ємів OpenIPC IP-камери (звірте з маркуванням саме вашої плати):

| Група | Контакти |
|---|---|
| **LAN / Ethernet** | TX+ (1), TX− (2), RX+ (3), RX− (6) |
| **Живлення** | GND, +12V |
| **Аудіо** | GND, In, Out |
| **Тривога (Alarm)** | GND, In |
| **Світлодіоди** | Green+, Amber+ |
| **Інше** | IRC (ІЧ-фільтр), D/N (день/ніч) |

::: warning Перед прошивкою
Для прошивки знадобляться контакти **UART (RX / TX / GND)**. На багатьох платах SSC30KQ вони винесені окремими пятачками — знайдіть їх за маркуванням `RX`/`TX`/`GND`.
:::

## Як прошити прошивкою OpenIPC

1. **Завантажте прошивку** для SSC30KQ з [релізів OpenIPC builder](https://github.com/OpenIPC/builder/releases/tag/latest) — шукайте файли виду `openipc.ssc30kq-nor-fpv.tgz` (для NOR-флеш) або `...-nand-...` (для NAND).
2. **Прошийте камеру** одним зі способів:
   - через **UART** — детальна [інструкція прошивки через UART](/configuration/uart-flash);
   - або за допомогою [Мультиконфігуратора](https://github.com/OpenIPC/openipc-configurator/releases) (простіше).
3. Для **APFPV**-прошивки (Wi-Fi напряму, без наземної станції) дивіться [гайд APFPV](/software/apfpv).

::: danger Увага
Не вимикайте живлення під час прошивки — це може "окирпичити" камеру.
:::

## SSC30KQ чи SSC338Q?

| | **SSC30KQ** | **SSC338Q** |
|---|---|---|
| Ядра | 1 | 2 |
| Сенсор | IMX335 (5 Мп) | IMX415 (8 Мп) |
| Макс. відео | 1080p\@60 / 720p\@120 | до 4K |
| Вага / ціна | менша | більша |
| Кому | легкі та бюджетні збірки | максимальна якість |

➡️ Якщо потрібно легко й дешево — **SSC30KQ**. Якщо потрібна максимальна картинка — [**SSC338Q + IMX415**](/hardware/vtx/sigmastar-ssc338q).
