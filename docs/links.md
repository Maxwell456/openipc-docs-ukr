# Корисні посилання

Перевірені ресурси для роботи з OpenIPC та цифровим FPV — офіційні джерела, активні спільноти й українські YouTube-канали.

## Інструменти

- [Таймер затримки відео (glass-to-glass)](/tools/latency-timer) — вимірювання наскрізної затримки відеолінка
- [Калькулятор бітрейту WFB-NG](/software/wfb-ng-config) — розрахунок відеобітрейту за MCS, шириною каналу та FEC

## Офіційні ресурси OpenIPC

| Ресурс | Опис |
| --- | --- |
| [openipc.org](https://openipc.org/) | Офіційний сайт проєкту OpenIPC |
| [GitHub OpenIPC](https://github.com/OpenIPC) | Початковий код, прошивки та релізи |
| [Telegram OpenIPC](https://t.me/OpenIPC) | Офіційний чат спільноти (англійською) |

## Українські спільноти

- [Є-дрон](https://t.me/e_drones) — спільнота з FPV та дронобудування
- [swarm.army](https://t.me/swarmalerts) — новини та сповіщення swarm.army

## YouTube-канали українською

- [E-Drone](https://www.youtube.com/@E-Drone) — огляди, збірки та налаштування
- [FPV питаннячка](https://youtube.com/@fpv-questions) — відповіді на типові питання новачків
- [Social Drone UA](https://youtube.com/@socialdroneua) — практичні гайди з FPV

## Магазини та каталоги

- [swarm.army](https://swarm.army/) — каталог FPV-комплектуючих за найкращими цінами
