Увага!

APFPV — це АЛЬФА-версія програмного забезпечення. Використовуйте на власний ризик. Ми не несемо відповідальності за можливу втрату керування через перешкоди або збої у зв'язку.

APFPV (Access Point FPV) — це максимально простий спосіб передачі цифрового відео з дрона на смартфон, планшет або комп'ютер через стандартний Wi-Fi. Ваш дрон стає точкою доступу, до якої ви підключаєтеся напряму, як до звичайного роутера.

INFO

Ця версія включає адаптивне потокове рішення a-link, розроблене Greg Sparks та Joakim, яке динамічно налаштовує якість відео на основі мережевих умов у реальному часі.

Для перегляду (наземна станція)VTX обладнання
• Android: додаток PixelPilot (рек.)
• Комп'ютер: будь-який веб-браузер
• Професійне: WiFi-обладнання (TP-Link, Ubiquiti)
• Radxa GS: перемикання WFB-ng / APFPV
• Будь-який пристрій з підтримкою RTP		• RTL8812AU або EU модулі
• RunCam WiFi Link V1/V2

• DIY SSC338Q (Thinker SD + NIC)

INFO

Для оптимальної продуктивності наполегливо рекомендується використовувати WiFi-модулі на базі RTL8812EU як на передавачі, так і на приймачі.

КРОК 1: Підключення VTX до інтернету

Підключіть ваш VTX до комп'ютера за допомогою кабелю Ethernet, USB-адаптера або UART-адаптера. Увімкніть пристрій та зачекайте 1-2 хвилини для повного завантаження системи.

Для завантаження прошивки VTX повинен мати доступ до інтернету через ваш домашній WiFi-роутер. Налаштування підключення виконується через SSH-з'єднання відповідно до інструкцій у наступному кроці.

КРОК 2: SSH-підключення до VTX

SSH (Secure Shell) дозволяє віддалено виконувати команди на вашому VTX з комп'ютера. Для встановлення з'єднання необхідно знати IP-адресу пристрою у вашій локальній мережі.

Встановлення з'єднання:

Windows
  1. Завантажте та встановіть PuTTY
  2. Запустіть програму та введіть IP-адресу VTX у полі "Host Name"
  3. Встановіть порт 22 та тип підключення SSH
  4. Натисніть "Open" для підключення
  5. Авторизуйтесь: логін root, пароль 12345
Mac / Linux

Відкрийте термінал та виконайте команду:

bash
ssh root@[IP-адреса-VTX]
1

Введіть пароль 12345 при запиті авторизації.

Як знайти адресу роутера

Адреса шлюзу відображається у системних налаштуваннях мережі як "Default Gateway" або "Шлюз за замовчуванням".

Windows: Відкрийте командний рядок та виконайте ipconfig. Знайдіть "Default Gateway".

Mac: Системні налаштування → Мережа → Додатково → TCP/IP → Router.

Linux: Виконайте ip route | grep default у терміналі.

Авторизуйтесь у панелі керування роутером. Стандартні облікові дані зазвичай вказані на наклейці роутера. Типові комбінації: admin/admin, admin/password, або admin/[порожнє поле].

Знайдіть розділ підключених пристроїв. Можливі назви розділу:

  • Connected Devices / Підключені пристрої
  • Device List / Список пристроїв
  • Attached Devices / Приєднані пристрої
  • DHCP Clients / DHCP-клієнти
  • LAN Settings / Налаштування LAN

Знайдіть пристрій з назвою Sigmastar — це ваш VTX. Скопіюйте відображену IP-адресу.

КРОК 3: Встановлення прошивки

Після успішного SSH-підключення та налаштування доступу до інтернету виконайте команду встановлення у консолі VTX:

bash
curl -L -o /tmp/openipc.ssc338q-nor-apfpv-greg08RC2.tgz \
https://github.com/sickgreg/OpenIPC_sickgregFPV_apfpv/raw/main/openipc.ssc338q-nor-apfpv-greg08RC2.tgz && \
sysupgrade --archive=/tmp/openipc.ssc338q-nor-apfpv-greg08RC2.tgz -f -n
1
2
3

Натисніть Enter для запуску процесу.

Процес встановлення:

[1/4] Завантаження прошивки...
[2/4] Перевірка цілісності...
[3/4] Встановлення системи...
[4/4] Перезавантаження...
1
2

Тривалість процесу: 5-10 хвилин. VTX автоматично перезавантажиться після завершення.

Важливо

Від'єднайте кабель Ethernet після завершення встановлення для коректної роботи відеопотоку.
