Waybeam Venc — новий відеоенкодер для FPV ​

waybeam venc — це новий автономний відеоенкодер з відкритим кодом (MIT) для камер на чіпах SigmaStar Infinity6E (Star6E / SSC338Q) та Infinity6C (Maruko), що повністю замінює Majestic у FPV-стримінгу.

🔹 Що нового? ​

  1. Повна заміна Majestic для WFB-ng:

    • Нативна інтеграція через абстрактний Unix-сокет (unix://wfb_tx) — без UDP-накладних витрат
    • Підтримка Shared Memory (shm://) для максимальної продуктивності
    • Кодек H.265 (HEVC) та H.264 з режимами CBR / VBR / AVBR / FIXQP

  2. HTTP API — 84 параметри в реальному часі:

    • Зміна бітрейту, FPS, роздільної здатності, балансу білого — без перезапуску
    • Вбудований веб-дешборд на порту 80: конфігурація, API-docs, ISP-тюнінг
    • BusyBox wget-сумісний: всі ендпоінти через GET-запити

  3. Gemini-режим — стрімінг + запис одночасно:

    • Стрімінг через WFB-ng у низькому бітрейті + запис на SD у високій якості (до 20 Мбіт/с, 120 fps)
    • Формат MPEG-TS: безпечний при раптовому відключенні живлення
    • Автоматичне зниження бітрейту запису якщо SD-карта не встигає

  4. GyroGlide-Lite — гіроскопна стабілізація зображення (Star6E):

    • BMI270 IMU з апаратним FIFO, синхронізованим з кадрами
    • Компенсація тремтіння через зсув вікна VPE-кроппера — без навантаження на CPU
    • Авто-калібрування гіроскопа при старті (2 секунди нерухомості)

  5. ROI-кодування для FPV:

    • Центр кадру кодується у вищій якості — гарантована чіткість там, де важливо

🔹 Як розпочати? ​

  • Огляд waybeam venc — всі можливості та порівняння з Majestic
  • Встановлення на камеру — завантаження бінарника, конфіг /etc/venc.json, перший запуск
  • Інтеграція з WFB-ng — замінити Majestic, налаштувати наземну станцію
  • Веб-панель та HTTP API — керування через браузер і командний рядок
