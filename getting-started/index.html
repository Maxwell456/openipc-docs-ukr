Швидкий старт ​

В OpenIPC все дуже швидко змінюється, тому ця документація буде з часом змінюватись:

Частина 1 – що купити, мінімальні налаштування та літати.
Частина 2 – для ентузіастів, які люблять тримати паяльник в руках, друкувати на 3D‑принтері та вносити корективи через SSH.

Частина 1: Що купити та літати ​

VTX (камери та передавачі)

  • Runcam Wifi Link v1 – IMX415 + SigmaStar SSC338Q, EU2 модуль (за потреби можна замінити на інший)

  • Runcam Wifi Link v2 – IMX415 + SSC338Q, EU2 модуль вбудований у плату

  • EMAX Wyvern Link – Доступний на AliExpress, підходить для TinyWhoop

  • OpenIPC Thinker v1 (без Wi‑Fi) – Лише плата, Wi‑Fi карту купувати окремо (наприклад RTL8812EU2)

  • OpenIPC Thinker v1 (з Wi‑Fi) – Вбудований Wi‑Fi передавач 100 mW (підходить лише для TinyWhoop, дальність до ~500 м)

Антени

  • Роз’єм UFL: Rush Cherry UFL‑LHCP Компактна антена для 2–3" дронів (2 виходи)
  • Поляризація: LHCP → LHCP або RHCP → RHCP

VRX (приймальні станції)

  • RunCam WiFiLink‑RX Digital HD Receiver
    Готове рішення: Radxa + 2 EU2 плати
    Під’єднується до окулярів або монітора
  • Emax Ground Station
    Тестові варіанти, поки не у продажу
  • PixelPilot (Android)
    Перетворює смартфон на наземну станцію (Wi‑Fi адаптер RTL8812AU)
    Плюси: дешевизна
    Мінуси: затримка ~50–70 ms, залежить від потужності телефону
  • Radxa Zero 3W Hat
    • Radxa Zero 3W + USB Hub + microSD + 2 шт мережевих картки (RTL8812EU2 або RTL8812AU) + 3D‑друкований корпус

Частина 2: Для ентузіастів своїми руками ​

Компоненти для Radxa Zero 3W Ground Station

  • Radxa Zero 3W (2 GB, без eMMC, з header)
  • Wi‑Fi карти (2 шт.)
    • RTL8812AU: компактні, дешеві, працюють на 3.3 V, до 40 MHz
    • RTL8812EU2: потужніші, потребують USB‑hub та 5 V живлення
  • BEC (понижуючі перетворювачі): 5 V і 3.3 V регульовані
  • USB HAT / хаб: для підключення кількох карт
  • Micro-USB → USB‑C кабель: для HAT
  • microSD карта: ≥ 64 GB (образ ~1.5 GB + DVR ~1 GB/10 хв)
  • HDMI‑mini HDMI кабель або micro‑HDMI→HDMI адаптер
  • 3D‑друк корпусу (PETG)
  • Вентилятори 25×25 мм
  • Кнопки/кінцевики для запису та AP‑режиму
  • XT60 (мама) конектор для живлення
  • Антени 4 шт. (2 спрямовані + 2 патчі, LHCP або RHCP)
  • RP‑SMA подовжувачі/перехідники

Приклади зібранної наземної станції

  • Варіант 1: USB‑HAT + 2×RTL8812AU → 227 g без антен

- *Варіант 2*: USB‑hub + 2×RTL8812EU → 105 g без антен
